Limbo Picman Cfsh Arc Projects

Cfsh

Cfsh is a shell that mimics the Bash terminal and allows you to traverse AWS components just like you would directories in a file system with "cd" and "ls".

Astar

A web UI to create mazes and patterns and then watch how the Astar algorithm finds the shortest path from the start to the end.

SpaceForm

The Earth suspended in mid air with magnets which can accurately rotate to match the current Earth's position. Or just set it to any time and watch it rotate.

SpaceX Delta V Map

See what the payload capacity is of SpaceX's fleet of rockets and watch the routes visualized on a delta V map of the Solar System.

Planets API

The planets API is an interface to simulate the movements of objects in space. The api takes in the state of a system of bodies in space and returns the new values after a certain period of time.

Game of Life

A web UI to create patterns and watch how Conway's Game of Life plays out. Save patterns and use the default patterns like snowflake.

Mine Sweeper

The classic Windows game Minesweeper. Also available as an Android app.
